# QLA to SRT

A Flask web app that turns a school **Question Level Analysis (QLA)** spreadsheet into targeted intervention resources — fast, without manually crunching the data.

Upload your QLA Excel file and optionally the original exam papers, and the app gives you:

- A full analysis dashboard with class rankings, topic heatmaps, and per-class breakdowns
- A Department Overview with priority starter targets and per-class focus cards
- AI-generated **SRT (Student Response Time)** question sheets — one Word doc per class, tailored to each group's weakest topics

---

## How it works

1. **Upload** — drop in the QLA `.xlsx` file and (optionally) the exam paper `.doc`/`.docx` files for each ability group
2. **Analyse** — the app parses every sheet, computes per-topic averages, standard deviations, and class rankings
3. **Review** — explore the analysis dashboard or the Department Overview to see where students are weakest
4. **Generate** — click *Generate Sheets* and the app calls the Claude AI API to write one targeted question per weak topic, formatted for the ability group, and saves them as Word documents ready to print

---

## Features

### Analysis dashboard (`/upload`)
- Cohort summary — total students, year group, class count, cohort average
- Cross-group weak topics — topics struggling across multiple ability groups, flagged as priority intervention targets
- Per-group class performance table with rank, average, and vs-group-average delta
- Colour-coded topic heatmap sorted worst-first (per group)
- Between-class variance — topics where classes diverge most
- Per-class breakdown cards showing weakest 3 and strongest 3 topics

### Department Overview (`/department`)
- Priority Starter Targets — cross-group weak topics ranked by worst average, ready to drop into lesson starters
- Class Starter Focus Cards — top 5 weakest topics per class, sorted weakest class first, filterable by ability group
- Full topic heatmap for each ability group
- Hover tooltips showing the actual exam question text (when papers are uploaded)
- Print-ready layout

### SRT Question Sheets (`/generate/srt`)
- One Word document per class
- One targeted question per weak topic (top 5 by default)
- Question style matched to ability group: short answer (Core Plus), multiple choice (Numeracy), exam style (Extension)
- Generated by Claude AI using the QLA data and, if uploaded, the original exam paper questions as source material

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/PressureJag/QLA-to-SRT-JBA.git
cd QLA-to-SRT-JBA
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Add your API key

```bash
cp .env.example .env
```

Open `.env` and add:

```
ANTHROPIC_API_KEY=sk-ant-...
SECRET_KEY=any-random-string
```

### 4. Run the app

```bash
python app.py
```

Then open `http://localhost:5050` in your browser.

---

## Input files

Place the following in the upload form on the home page:

| File | Required | Notes |
|---|---|---|
| QLA spreadsheet | Yes | `.xlsx` — the Analysis Grid exported from your MIS |
| Exam papers | No | `.doc` or `.docx` — one per ability group; unlocks real question text in the dashboard and tooltips |

The app auto-detects ability groups (Core Plus, Numeracy, Extension) from the sheet names in the QLA file.

---

## Output files

Generated Word documents are saved to the `output/` folder and linked for download on the results page. You can download individually or as a single zip.

```
output/
  SRT_Year7_7CP1_2026-06-08.docx
  SRT_Year7_7CP2_2026-06-08.docx
  SRT_Year7_7N1_2026-06-08.docx
  ...
```
